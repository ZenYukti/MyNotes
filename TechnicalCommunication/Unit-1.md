### 5.2 Code (How to Say)

**Definition**: The form, language, style used to convey message

**Elements:**
- Language choice
- Formality level
- Tone
- Medium
- Structure

**Examples:**

**Formal Code:**
```
"Dear Sir/Madam,

I would like to formally request a leave of absence for 
three days, from January 10-12, 2026, due to personal 
reasons. I will ensure all pending work is completed 
before my leave.

I would be grateful if you could approve my request.

Yours sincerely,
John Doe"
```

**Informal Code:**
```
"Hey!

Can I take Jan 10-12 off? Got some personal stuff. 
I'll finish everything before leaving.

Thanks!
John"
```

**Same content, different code!**

### 5.3 Content (What to Say)

**Definition**: The actual message, information, ideas

**Example:**
```
Content: "Leave request for 3 days"

Different Codes:
1. Email (written)
2. Face-to-face (oral)
3. Phone call (oral)
4. Leave form (written)

All convey same content, different codes
```

### 5.4 Matching Code to Context

| Situation | Appropriate Code | Why |
|-----------|------------------|-----|
| **Job Application** | Formal written | Professional setting |
| **Friend's Birthday** | Informal oral | Personal relationship |
| **Emergency** | Quick, direct | Urgency |
| **Legal Notice** | Formal written | Legal validity |
| **Team Chat** | Informal written | Quick coordination |

**Example Scenario:**

```
Content: "Meeting at 3 PM"

Context 1: CEO to Board Members
Code: "Dear Board Members, kindly note that the quarterly 
      meeting is scheduled for 3:00 PM today in the 
      boardroom. Your presence is highly appreciated."

Context 2: Friend to Friend
Code: "Hey! Meeting at 3. See you there!"

Context 3: Automated System
Code: "REMINDER: Meeting scheduled at 15:00 today."
```
